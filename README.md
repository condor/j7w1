# J7W1

A gem to send the push notification to mobile app via Amazon Simple Notification Service (SNS)

## Installation

Add this line to your application's Gemfile:

    gem 'j7w1'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install j7w1
